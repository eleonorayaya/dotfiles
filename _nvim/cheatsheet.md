## Windows

### Split Vertically

```
Cmd+K
```

### Navigate Splits

Directions are `h` for left, `j` for down, `k` for up, and `l` for right

```
space+w+direction
```

## Folds

### Fold all

```
space+z+m
```

### Unfold all

```
space+z+r
```

### Toggle fold

```
space+z+z
```

### Open fold

```
space+z+o
```

### Close fold

```
space+z+c
```

## LSP

### Go to definition

```
space+gd
```

### Go to type definition

```
space+gt
```

### Go to implementation

```
space+gi
```

### Go to references

```
space+gr
```

### Go to declaration

```
space+gD
```

### Show documentation

```
space+K
```

### Show type information

```
space+K
space+t
```
